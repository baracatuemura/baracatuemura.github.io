# baracatuemura.github.io

Personal portfolio for **Guilherme Baracat Uemura** — built with React + Vite.

## Dev

```bash
npm install
npm run dev      # local dev server
npm run build    # production build -> dist/
npm run preview  # preview production build
```

## Deploy

Pushes to `master` trigger a GitHub Actions workflow that builds the project and deploys to GitHub Pages.
